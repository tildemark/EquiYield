# Changelog

## v1.0.0 (2026-01-09)
- Initial release of EquiYield scaffold (Express API, Prisma, Redis, Next.js admin/member UI).
- Admin/member auth flows, loan and contribution management, dividend payouts, and transaction ledger for members.
- Admin loan editing gated to loans without payments and not marked PAID.
- Co-maker visibility with payment status for both admin and member dashboards.
